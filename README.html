DungeonEngine

A custom C++ game engine and game targeting Nintendo Switch and PC. Inspired by Barony (visual style) and Hellgate: London (gameplay feel).

Building

cmake -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build
./build/dungeon_game
